This project analyzes employee attrition trends and workforce distribution using Excel and Power BI. The dashboard provides insights into employee turnover, department performance, and location-based workforce patterns.
The objectives of this project is analyze employee attrition rate, identify high-risk departments, evaluate workforce distribution, support HR decision-making with data insights.
Tools Used are Microsoft Excel (Data Cleaning & Structuring), Microsoft Power BI (Data Modelling & Dashboard), DAX for KPI Calculations.
Insights are identified departments with higher attrition, location-based workforce distribution analyzed and attrition trends based on tenure observed.
